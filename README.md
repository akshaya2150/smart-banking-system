# 💳 Smart Banking System

A full-stack banking simulation built using **Spring Boot** and **HTML, CSS, JavaScript**.

---

## 🚀 Features

* User Registration & Login
* Create multiple accounts (Savings, Current, Student)
* Account limits:

  * Savings → 2
  * Current → 2
  * Student → 1
* Deposit, Withdraw, Transfer (Account → Account)
* PIN-based secure balance check
* Multi-account dashboard (card UI)
* Transaction history with search
* Last transaction preview
* Dark / Light mode

---

## 🛠 Tech Stack

* Backend: Java, Spring Boot, JPA, MySQL
* Frontend: HTML, CSS, JavaScript

---

## ⚙️ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/smart-banking-system.git
```

* Run backend (Spring Boot)
* Open `index.html`
* Backend runs on: `http://localhost:8081`
